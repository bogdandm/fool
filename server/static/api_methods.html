Все методы возращают json-файл вида: 
{ data: [< Change >, ...]}
< Change > = {
	"card": Card,
	"inf": int,
	"player": 0|1,
	"type": < Type >
}
< Type > = player_switch | take_cards | get_card | trump_card | attack | defense | table_clear

  • api.attack(card: int)
  • api.defense(card: int)
